## 2026-06-16 15:35 - ISRO 返回 UI 界面添加成功的方法

本次已确认成功的只限于 UI 显示层，不代表功能链路全部成功。

- 真实右键入口仍然是 `SetMsgBoxHandler` 的 `arg0=0x21`，不能再从 `0x2A/0x2B` 入手。
- 成功显示 ISRO 样式返回界面的关键做法是：
  - `0x007BAAD9` 右键返回卷轴调用跳到 `0x012EC930` field bridge stub。
  - stub 只处理 `0x21/0x25/0x28`，并把 CSRO 小返回框字段桥接到五/六按钮 UI 需要的字段。
  - stub 在进入 `0x012DFA80` 前补齐关键字段：保存 item slot 到 `0x012ECFF0`，桥接 `[ecx+0x808]/[ecx+0x80C]/[ecx+0x810]/[ecx+0x854]`，避免旧的空 `[esi+0x810]` 崩溃。
  - `0x21` 和 `0x25` 的大分发表项指向 `0x012E39EF`，由 CSRO 现有 `.rret` 五/六按钮分支创建 ISRO 返回 UI。
- 这套方法的成功边界：界面能显示出 ISRO 风格的返回按钮；它不证明旧功能、新增功能、服务端链路已经全部正确。
